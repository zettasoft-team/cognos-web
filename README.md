# Zmeta Web

데이터 모델 XML 파일을 업로드하고 테이블·ERD·차원맵으로 시각화하는 메타데이터 관리 플랫폼입니다.
Vite + React 기반 SPA로, 별도의 FastAPI 백엔드 서버와 연동하여 동작합니다.
사용자 인증, 문서 관리, 보고서, 요청 게시판, 관리자 기능을 제공합니다.

---

## 로컬 실행

```bash
# 의존성 설치 (최초 1회)
npm install

# 개발 서버 실행
npm run dev
```

또는 `run.bat`을 더블클릭하면 자동으로 환경 확인 후 개발 서버가 시작됩니다.

> 백엔드 서버가 실행 중이어야 API 연동이 됩니다. API 주소는 `.env` 파일의 `VITE_API_BASE_URL`로 설정합니다.

### CI
현재 CI는 구현되어 있지 않습니다.
`/home/zetta/zmeta-web`에서 `git pull origin main` 진행해주세요.

### CD
scripts 폴더에서 아래 쉘을 가져와 실행 시 docker 빌드 및 배포됩니다.

```sh

```

## 화면 목록 및 기능

### 로그인

- 이메일 / 비밀번호 로그인
- 로그인 상태 유지 (미구현)
- 비밀번호 찾기 (미구현)

---

### 메타 문서 관리 — 목록

- XML 파일 다중 업로드
- 문서 목록 조회 (페이징)
- 파일명 · 프로젝트명 · 설명으로 검색
- 설명 인라인 편집
- 단건 삭제 / 선택 일괄 삭제

---

### 메타 문서 관리 — 상세

- 프로젝트 트리 탐색기 (좌측 패널)
- 탐색기 탭 — 선택 테이블의 컬럼 정보 (논리명 · 물리명 · 타입 · PK · NULL)
- ERD 다이어그램 탭 — 테이블 관계 시각화
- 차원맵 탭 — 차원 / 계층 / 레벨 구조 표시
- Excel 다운로드 / 업로드
- 원본 XML 다운로드

---

### 보고서 관리(미구현)

- 보고서 목록 조회 (페이징)
- 제목 · 작성자 검색
- 선택 일괄 삭제
- 상태 표시 (게시됨 · 임시저장 · 검토중)
- 새 보고서 작성 (미구현)
- 내보내기 (미구현)

---

### 요청 게시판(미구현)

- 요청 목록 조회 (페이징)
- 카테고리 탭 필터 (전체 · 아이디어 · 버그)
- 상태 토글 필터 (대기 · 진행중 · 완료 · 반려)
- 분야 · 우선순위 드롭다운 필터
- 글 작성 (미구현)
- 상세 보기 · 댓글 (미구현)

---

### 관리자(미구현)

**사용자 관리**
- 사용자 목록 조회 · 검색
- 비밀번호 초기화 (임시 비밀번호 발급)
- 활성 / 비활성 상태 변경
- 사용자 삭제
- 사용자 추가 · 편집 (미구현)

**권한 관리**
- 역할 목록 조회
- 역할 삭제
- 역할 추가 · 편집 (미구현)

**모델 지원 데이터**
- 데이터소스 · 카테고리 · 태그 목록 조회 · 검색
- 항목 삭제
- 항목 추가 · 편집 (미구현)
